# Sword Project

This repository contains a small web project with a shop interface and database support.

## Files

- `index.html` - Main landing page for the project.
- `styles.css` - Site styling and layout rules.
- `script.js` - Client-side JavaScript functionality.
- `create_database.php` - PHP script to create the database tables.
- `db_connect.php` - PHP database connection helper.
- `shop/` - Folder for shop-related assets and pages.
- `sword wear.webp` - Image asset used in the project.

## Setup

1. Place the project in a PHP-enabled web server root.
2. Configure `db_connect.php` with your database credentials.
3. Run `create_database.php` once to initialize the database.
4. Open `index.html` in the browser through the server to view the site.
